- Three-column layout (3:6:3 ratio)
- Full weekly grid visible
- All features accessible

### Tablet (768px - 1199px)
- Collapsible sidebars
- Scrollable weekly grid
- Touch-friendly controls

### Mobile (< 768px)
- Stacked layout
- Swipeable sections
- Bottom sheet modals

## Accessibility Features

- Keyboard navigation support
- ARIA labels on interactive elements
- Focus indicators on all controls
- Color contrast meets WCAG AA standards
- Screen reader announcements for progress updates

## Tips for Best Experience

1. **Start with Recipe Pool**: Browse and add recipes you want to cook
2. **Adjust Servings**: Use the modal to set exact serving counts
3. **Review in Staging**: Check your selections before assigning
4. **Use Auto-Fill**: Let the system distribute meals intelligently
5. **Manual Override**: Drag or click to adjust specific days
6. **Track Progress**: Watch the bars fill as you plan
7. **Celebrate**: Enjoy the animations when sections complete!
